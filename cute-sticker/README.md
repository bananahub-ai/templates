# Cute Chibi Sticker Pack

Official prompt template for creating cute chibi sticker or emoji-style assets with clean outlines and simple expressions.

## Install

```bash
bananahub add bananahub-ai/templates/cute-sticker
```

## Verified Models

- `gemini-3.1-flash-image-preview` — verified with `samples/sample-3.1-flash-01.png`

## Supported Models

- `gemini-3-pro-image-preview` — best fit for cleaner rendering and stronger consistency
- `gemini-3.1-flash-image-preview` — good fast option for simple stickers
- `gpt-image-2` — supported through the `gpt-image` prompt variant, pending sample verification
- `gpt-image-1` — compatible fallback through the `gpt-image` prompt variant
