# Cute Chibi Sticker Pack

Official prompt template for creating cute chibi sticker or emoji-style assets with clean outlines and simple expressions.

## Install

```bash
bananahub add bananahub-ai/templates/cute-sticker
```

## Verified Models

- `gpt-image-2` — verified with `samples/sample-gpt-image-2-01.png`

## Supported Models

- `gpt-image-2` — supported through the `gpt-image` prompt variant for clean, centered sticker generation
- `gpt-image-1` — compatible fallback through the `gpt-image` prompt variant

## Sample Outputs

| File | Model | Prompt Variant | Notes |
|---|---|---|---|
| `samples/sample-gpt-image-2-01.png` | `gpt-image-2` | `gpt-image` | Chubby orange cat sticker with plain white background and clean die-cut silhouette |
