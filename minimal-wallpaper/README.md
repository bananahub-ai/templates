# Minimal Phone Wallpaper

Official prompt template for minimal mobile wallpapers with clear subject, calm composition, and generous negative space.

## Install

```bash
bananahub add bananahub-ai/templates/minimal-wallpaper
```

## Verified Models

- `gpt-image-2` — verified with `samples/sample-gpt-image-2-01.png`

## Supported Models

- `gpt-image-2` — supported through the `gpt-image` prompt variant for clean negative-space wallpaper compositions
- `gpt-image-1` — compatible fallback through the `gpt-image` prompt variant

## Sample Outputs

|---|---|---|---|
