# Cyberpunk City Nightscape

Official prompt template for cinematic neon city scenes. This is a gallery-style creative template, so it lives in the remote official library instead of the BananaHub Skill starter pack.

## Install

```bash
bananahub add bananahub-ai/templates/cyberpunk-city
```

## Verified Models

- `gpt-image-2` — verified with `samples/sample-gpt-image-2-01.png`

## Supported Models

- `gpt-image-2` — supported through the `gpt-image` prompt variant for cinematic neon city scenes
- `gpt-image-1` — compatible fallback through the `gpt-image` prompt variant
